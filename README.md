## Usage

#### The Web app provides following functionalities :
1. Registration form with minimum details in order to know the details of the person
getting enrolled.
   - Name
   - Status: Female / Kid / Senior Citizen
   - DOB
   - Address
   - Profile
   - Emergency Contact Numbers upto 4
